Package.json generated:
npm init -y

Express server manager install:
npm i express

mongodb database access install:
npm i mongoose

.env environment manager file:
npm i dotenv

hbs handlebars for syntax moustaches curly brace use:
https://www.npmjs.com/package/handlebars
npm i hbs

nodemon auto restart server on save file:
npm i --save nodemon

Morgan error logging:
npm i morgan

joi used to validate data input to forms:
npm install @hapi/joi

bcrypt to hash the passwords
npm install --save bcrypt

JsonWebToken library
npm install jsonwebtoken
